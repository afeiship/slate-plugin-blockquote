> h1/h2/h3/h4/h5/h6 tags are not used for headings in Tailwind CSS. Instead, use the `className` prop to apply the appropriate font-weight class.

```jsx live
<div className="x-5 justify-center">
  <section className="relative debug-green size-50">
    <div className="absolute layout-abs-center debug-red fcc w-25 h-6 bg-slate-400">
      Small Box
    </div>
  </section>

  <section className="relative debug-green size-50">
    <div className="absolute layout-abs-center-x debug-red fcc w-25 h-6 bg-slate-400">
      Small Box - x
    </div>
  </section>

  <section className="relative debug-green size-50">
    <div className="absolute layout-abs-center-y debug-red fcc w-25 h-6 bg-slate-400">
      Small Box - y
    </div>
  </section>
</div>;
```
