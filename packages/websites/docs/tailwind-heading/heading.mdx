> h1/h2/h3/h4/h5/h6 tags are not used for headings in Tailwind CSS. Instead, use the `className` prop to apply the appropriate font-weight class.

```jsx live
<div className="bg-white rounded-md y-2 p-4">
  <h1>道可道，非常道；名可名，非常名。</h1>
  <h2>無名天地之始；有名萬物之母。</h2>
  <h3>故常無欲以觀其妙，常有欲以觀其徼。</h3>
  <h4>此兩者，同出而異名，同謂之玄。</h4>
  <h5>玄之又玄，众妙之门。</h5>
  <h6>學而不思則罔，思而不學則殆。不知所學，焉能知所不學？</h6>
</div>
```
